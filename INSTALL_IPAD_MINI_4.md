# iPad mini 4 安裝及長期顯示

## 1. 更新 iPad

1. 打開「設定」→「一般」→「軟件更新」。
2. iPad mini 4 應更新至裝置提供的最新 iPadOS 15 版本。
3. 更新完成後重新啟動 iPad。

## 2. 加入主畫面

1. 將 iPad 打橫。
2. 用 Safari 開啟：<https://jimoswang.github.io/home-dashboard/>
3. 按 Safari 的分享按鈕。
4. 選「加入主畫面」。
5. 名稱保留「屋企資訊板」，然後按「加入」。
6. 回到主畫面，由新圖示開啟 Dashboard。

必須使用 Safari 完成「加入主畫面」，不要用其他瀏覽器建立捷徑。

## 3. 設定外母大人的 Profile

1. 長按 Dashboard 左上角時間 3 秒。
2. 首次進入 Settings 時設定 6 位 Admin PIN。
3. 選擇天氣地區，再加入巴士路線、方向及站牌。
4. 如已有另一部裝置的設定，可在原裝置 Export JSON，再於 iPad Import JSON。
5. Admin PIN 不會包含於 Export JSON，需要在 iPad 另外設定。

## 4. 長期顯示

1. 「設定」→「顯示與亮度」→「自動鎖定」→「永不」。
2. 「設定」→「輔助使用」→「引導使用模式」→開啟，並設定密碼。
3. 開啟 Dashboard 後連按三次 Home 鍵，啟動「引導使用模式」。
4. 長期接駁合規格充電器，並避免陽光直射及過熱。

iPad 網頁無權自行關閉或喚醒螢幕，因此 Android／Fully Kiosk 的定時熄屏功能不適用。夜間可手動按電源鍵熄屏；再次喚醒時會返回原本 Dashboard。

## 5. 更新及離線

- 有網絡時，網站會自動取得新版。
- 巴士、天氣或雷達個別失靈不會令整頁白屏。
- 短暫離線會顯示最後成功快取資料；恢復連線後自動更新。
